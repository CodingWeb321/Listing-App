This repository contains the front-end implementation for a comprehensive job listing platform.

## Features Implemented

- **Job Listing Display:** Shows key job details including job title, company name, location, salary, and date posted.
- **Search Functionality:** search allows finding jobs by multiple keywords (AND logic) across job title, company name, and location.
- **Multi-Filter System:** Users can filter jobs based on job type, employment type, and seniority level.
- **CRUD Operations:** Full functionality for Adding, Editing, and Deleting job listings.
- **Modular Design:** Components are built to be reusable wherever possible

## Technology Stack

- **Frontend Framework:** React.js
- **State Management:** React Hooks (`useState`, `useEffect`)
- **API/Data:** Simulated REST API using JSON Server for persistence.
- **Icons:** **React Icons** React icons for high-quality, scalable SVG image for logo.
